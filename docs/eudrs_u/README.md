# EUDRS-U Documentation

Research handoff and specification scaffolding for the EUDRS-U v1.0 initiative.

## Files

- `EUDRS_U_v1_0_SCIENTIST_HANDOFF.md`: Repo-anchored implementation handoff for research scientists.
- `EUDRS_U_v1_0_SPEC_OUTLINE.md`: Structured template for a repo-compatible formal spec.

## Purpose

- Align research/spec work with existing repository constraints (determinism, trust boundaries, artifact contracts).
- Reduce translation overhead between research outputs and implementation teams.

## Usage

- Start with the handoff to understand current implementation surface.
- Use the spec outline to produce a normative artifact inventory and acceptance criteria.
